# Woolang project generator

> This is a part of [woolang](https://github.com/woo-lang/woolang)

## Installation
Make sure you have pip and python
`python -v`
`pip -v`

Install the package 
`pip install [woolang-project-generator](https://pypi.org/project/woolang-project-generator/)`

## How to use it

```python
from woolang_project.project import Project

project = Project()
```
